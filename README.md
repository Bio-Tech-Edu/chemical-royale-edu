## 📖 Sobre o Projeto
O **Chemical Royale: A Arena dos Elementos** é um ambiente virtual de aprendizagem inovador e autoinstrucional, focado no ensino de Química para estudantes do Ensino Médio. Inspirado na mecânica de escolha de heróis e batalhas estratégicas, o sistema transforma a resolução de questões do ENEM em uma experiência imersiva através do uso de metodologias ativas (Gamificação e Storytelling).

Projeto coordenado pela **Prof.ª Pâmella** sob a chancela educacional da **BIO+TECH EDUDESIGN**.

## 🚀 Funcionalidades
- **Autenticação de Jogador:** Registro de estudantes via e-mail educacional.
- **Narrativa Interativa (Branching Scenario):** Escolha entre 3 versões de storytelling (A Jornada do Herói, In Media Res, Investigativo/Mistério).
- **Conteúdo Gamificado (H5P):** Aulas interativas cobrindo Estrutura Atômica, Tabela Periódica, Ligações Químicas e Funções Inorgânicas.
- **Progressão e Desafios:** Blocos de 3 questões modelo ENEM com níveis de dificuldade progressiva (Fácil, Médio, Difícil).
- **Geração Automática de Relatórios:** Emissão de PDF consolidado com o desempenho e autoavaliação do aluno, enviado automaticamente para a coordenação.

## 🛠️ Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3, JavaScript
- **Interatividade Educacional:** Ecossistema H5P (Course Presentation, Branching Scenario, Single Choice Set, Dialog Cards, Question Set)
- **Hospedagem:** GitHub Pages
- **Relatórios:** Bibliotecas JS para geração de PDF (ex: jsPDF)

## ⚙️ Instalação Local (Para Desenvolvedores)
1. Clone o repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/chemical-royale-edu.git
   ```
2. Abra a pasta do projeto:
   ```bash
   cd chemical-royale-edu
   ```
3. Inicie um servidor local (ex: usando a extensão Live Server do VSCode ou Python):
   ```bash
   python -m http.server 8000
   ```
4. Acesse `http://localhost:8000` no seu navegador.
